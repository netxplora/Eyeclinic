# Satome Eye Clinic

Professional eye care services website for Satome Eye Clinic, Benin City, Edo State.

## Features

- **Appointment Booking**: Patients can schedule appointments online.
- **Admin Dashboard**: Comprehensive management system for staff.
- **Real-time Updates**: Live tracking of appointments and requests.
- **Waitlist System**: Automated waitlist management for fully booked days.
- **Blog Section**: Health education and clinic updates.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Shadcn UI
- **Backend**: Supabase (Database, Auth, Edge Functions)
- **Email**: Resend API

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Set up environment variables in a `.env` file:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
